Features

Built-in features for SEO, search, and more

Hwaro includes powerful features out of the box.

Built-in Features

  • SEO — Sitemaps, RSS feeds, robots.txt, OpenGraph, and Twitter Cards
  • Search — Client-side search index generation (Fuse.js compatible)
  • Syntax Highlighting — Code block highlighting with Highlight.js
  • Pagination — Split large content lists into multiple pages
  • Multilingual — Build multilingual sites with translation linking and hreflang tags
  • Markdown Extensions — Task lists, footnotes, definition lists, math, and Mermaid diagrams
  • Auto Includes — Automatically load CSS and JS files from static directories
  • Content Files — Publish non-Markdown files from content directories
  • Cache Busting — Content-hash based browser cache invalidation for CSS/JS resources
  • Build Hooks — Run custom shell commands before and after builds
  • Incremental Build — Only rebuild changed files for faster builds on large sites
  • Streaming Build — Reduce memory usage for large sites with batch rendering
  • Asset Pipeline — Built-in CSS/JS bundling, minification, and fingerprinting
  • LLMs.txt — Generate instruction files for AI/LLM crawlers
  • Environment Variables — Reference environment variables in config and templates
  • Environment-Specific Config — Override config settings per deployment environment
  • PWA — Progressive Web App support with manifest.json and service worker
  • AMP — Generate AMP-compliant versions of content pages
  • Structured Data — Extended Schema.org JSON-LD (FAQ, HowTo, WebSite, Organization)
  • Auto OG Images — Auto-generate Open Graph preview images from page titles
