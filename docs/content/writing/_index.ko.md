콘텐츠는 `content/` 디렉터리에 둡니다. TOML(`+++`), YAML(`---`), JSON(`{...}`) 프론트 매터를 붙인 마크다운으로 작성합니다. 기본은 TOML이고 YAML도 널리 권장되며, JSON은 선호하는 사용자를 위해 지원합니다.

## 콘텐츠 유형

| 유형 | 파일 | URL |
|------|------|-----|
| 페이지 | about.md | /about/ |
| 섹션 | blog/_index.md | /blog/ |
| 섹션 페이지 | blog/post.md | /blog/post/ |

## 기본 콘텐츠 파일

```markdown
+++
title = "My Page"
date = "2024-01-15"
+++

Content in **Markdown**.
```

## 시작 순서

**페이지**에서 콘텐츠 파일과 프론트 매터의 기본을 먼저 익힙니다. 이어서 **섹션**을 읽고 블로그나 문서처럼 콘텐츠를 그룹으로 묶는 방법을 배웁니다. 그다음에는:

- **택소노미** — 태그, 카테고리, 커스텀 그룹으로 페이지 분류
- **숏코드** — 마크다운 안에 재사용 가능한 컴포넌트(YouTube, 알림, 갤러리) 삽입
- **아키타입** — `hwaro new`가 콘텐츠를 빠르게 스캐폴드하도록 템플릿 정의
