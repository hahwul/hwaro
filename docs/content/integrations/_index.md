Hwaro is built to play well with the rest of your toolchain — especially AI
coding agents. The classified error codes, `--json` output, and `AGENTS.md`
generation all exist so an agent can drive Hwaro reliably.

This section collects drop-in integrations.

## In This Section

- **[Agent Skills](/integrations/skills/)** — installable `SKILL.md` files that
  teach Claude Code, Cursor, OpenCode, Codex, and other skill-aware agents how
  to operate the Hwaro CLI and how to design great Hwaro sites.
