## Which scaffold should I use?

Use `hwaro init` with a scaffold:

```bash
hwaro init my-site --scaffold simple
hwaro init my-blog --scaffold blog
hwaro init my-docs --scaffold docs
```

- **simple**: marketing pages / small sites
- **blog**: posts + tags/categories
- **docs**: documentation-style structure
