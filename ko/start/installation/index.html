설치

Hwaro는 Crystal로 작성되었습니다. 소스에서 직접 빌드하거나 사전 빌드된 바이너리로 설치하면 됩니다.

Homebrew

brew tap hahwul/hwaro
brew install hwaro

Snapcraft

sudo snap install hwaro

APK (Alpine Linux)

최신 릴리스에서 .apk 패키지를 내려받아 설치합니다:

apk add --allow-untrusted hwaro-*.apk

DEB (Debian/Ubuntu)

최신 릴리스에서 .deb 패키지를 내려받아 설치합니다:

sudo dpkg -i hwaro_*_amd64.deb

RPM (Fedora/RHEL/CentOS)

최신 릴리스에서 .rpm 패키지를 내려받아 설치합니다:

sudo rpm -i hwaro-*.x86_64.rpm

AUR (Arch Linux)

yay -S hwaro

Nix

설치

nix profile install github:hahwul/hwaro

설치 없이 실행

nix run github:hahwul/hwaro -- --version

개발 셸

nix develop github:hahwul/hwaro

사전 빌드 바이너리

macOS와 Linux용 사전 빌드 바이너리를 GitHub Releases 페이지에서 받을 수 있습니다.

  1. 최신 릴리스에서 플랫폼에 맞는 바이너리를 내려받습니다.
  2. PATH에 포함된 디렉터리로 바이너리를 옮깁니다.
# Linux(amd64) 예시
chmod +x hwaro-v*-linux-x86_64
sudo mv hwaro-v*-linux-x86_64 /usr/local/bin/hwaro

소스 빌드

사전 요구 사항

  • Crystal 1.19+
  • Git

빌드

git clone https://github.com/hahwul/hwaro
cd hwaro
shards install
shards build --release --no-debug -Dpreview_mt

바이너리는 ./bin/hwaro에 생성됩니다.

-Dpreview_mt는 Crystal의 멀티스레드 런타임을 켭니다. 공식 바이너리와 Docker 이미지가 이 방식으로 빌드되며, hwaro build의 병렬 페이지 렌더링도 이 플래그가 있어야 동작합니다. 패키저는 항상 이 플래그를 넣어야 합니다. 없으면 빌드가 단일 스레드로 실행됩니다.

PATH 등록 (선택)

# PATH에 포함된 디렉터리로 복사
sudo cp ./bin/hwaro /usr/local/bin/

# 또는 bin 디렉터리를 PATH에 추가
export PATH="$PATH:$(pwd)/bin"

설치 확인

hwaro --version
