Crystal로 만든 정적 사이트 생성기. 테마 없이, 종속 없이. 내 콘텐츠와 구조, 속도 그대로.

시작하기

1,000페이지, 1초 안에.

공개 SSG 벤치마크의 minimal 워크로드(순수 마크다운 → HTML) 기준 콜드 빌드 중앙값입니다.

0.00s

막대는 실제 측정 시간과 같은 443ms 동안 채워집니다 — 전체 재빌드 한 번의 시간입니다.

  • minimal 마크다운 → HTML 0.44s
  • blog 태그, 페이지네이션, 피드 0.59s
  • heavy 구문 강조까지 1.50s

필요한 건 전부 내장.

플러그인을 찾아 헤맬 필요가 없습니다. config.toml에서 켜고 빌드하면 됩니다.

built_in_seo

사이트맵, 피드, OpenGraph, JSON-LD, llms.txt까지. 소셜 카드 이미지는 빌드 시점에 19가지 스타일로 렌더링됩니다.

blazing_fast

병렬 렌더링, 증분 빌드, 초대형 사이트를 위한 스트리밍 모드. 수천 페이지도 몇 초면 끝납니다.

markdown_first

TOML 프론트 매터, 각주, 체크리스트, 수식, Mermaid를 지원합니다.

+++
title = "Hello"
tags = ["crystal"]
toc = true
+++

flexible_templates

상속, 필터, 함수, 숏코드를 갖춘 Jinja2 템플릿. 테마에 갇히지 않고 직접 만듭니다.

taxonomies

태그, 카테고리, 커스텀 택소노미와 자동 생성되는 색인 페이지, 페이지네이션.

extensible

빌드 훅, 에셋 파이프라인, 이미지 처리, 다국어, PWA, AMP까지. 바이너리 하나에 전부.

스캐폴드에서 시작

다섯 가지 완성된 출발점이 토큰 시스템 하나를 공유하고, 라이트와 다크에 알아서 맞춰집니다.

빌드 타임에 그리는 소셜 카드

브라우저나 외부 서비스 없이 모든 페이지에 Open Graph 이미지가 만들어집니다. 19가지 스타일은 자동 OG 이미지 문서에서 확인할 수 있습니다.
