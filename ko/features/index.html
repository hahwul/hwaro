기능

SEO, 검색 등 기본 제공 기능

Hwaro는 플러그인 없이도 폭넓은 기능을 기본으로 제공합니다. 모든 기능은 config.toml에서 설정하며, 활성화하면 바로 동작합니다.

한눈에 보기

SEO & 탐색 — 사이트맵, RSS/Atom 피드, robots.txt, OpenGraph, Twitter 카드, 구조화된 데이터(JSON-LD), 자동 생성 OG 이미지.

콘텐츠 — 구문 강조, 검색 인덱스 생성, 페이지네이션, 마크다운 확장(각주, 작업 목록, 수식, mermaid), 시리즈, 관련 글, 이름 있는 내비게이션 메뉴.

성능 — 증분 빌드, 대규모 사이트를 위한 스트리밍 빌드, 압축(minify)과 핑거프린트를 지원하는 에셋 파이프라인, 반응형 변형과 LQIP 플레이스홀더를 만드는 이미지 처리, 캐시 버스팅.

파일 & 환경 — 환경 변수, 환경별 설정 오버라이드, CSS/JS 자동 인클루드, 콘텐츠 파일 게시.

플랫폼 — 다국어 지원, 이미지 처리, PWA, AMP, 여러 대상으로의 기본 제공 배포.
